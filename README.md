# Authentication UI

This project is a simple Authentication UI that includes both Login and Signup pages. It is built using HTML and CSS and focuses on clean design and layout.

## Features
- Login form with email and password
- Signup form with multiple input fields
- Side-by-side layout using Flexbox
- Clean and modern UI design
- Responsive alignment and spacing

## Tech Used
- HTML
- CSS

## What I Learned
- Flexbox for layout
- Form styling
- UI design structure
- Building real-world components
